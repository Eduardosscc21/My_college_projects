# 📖 Descripción

Este proyecto fue desarrollado como trabajo final para la materia de Minería de Datos. El objetivo principal consiste en aplicar técnicas de preparación, análisis y exploración de datos utilizando Orange Data Mining para identificar patrones relevantes dentro del conjunto de datos analizado.

El flujo de trabajo implementa distintas etapas del proceso de minería de datos, incluyendo selección de atributos, limpieza de información, detección de valores atípicos y generación de conjuntos de datos especializados para distintos análisis.

## 🎯 Objetivos del proyecto

* Analizar conjuntos de datos mediante técnicas de minería de datos.
* Identificar patrones y relaciones significativas dentro de la información.
* Realizar procesos de limpieza y preparación de datos.
* Detectar y tratar valores atípicos (Outliers).
* Generar subconjuntos de datos para análisis específicos.
* Aplicar técnicas de clasificación sobre los datos procesados.
* Facilitar la interpretación de resultados mediante herramientas visuales.

## 🎓 Contexto académico

Antes de ejecutar el proyecto asegúrate de tener instalado:
