## 📖 Descripción

Este proyecto fue desarrollado para la materia de ingeniería de software. El sistema es una aplicación web para la gestión integral de un gimnasio, permitiendo a diferentes tipos de usuarios (Usuario, Entrenador, Premium) acceder a funcionalidades específicas según su rol.

### 🎯 Objetivos del sistema

- Gestión de usuarios: Tres roles con accesos diferenciados
- Rutinas de ejercicio: Tren superior, inferior y tronco
- Seguimiento de progreso: Registro de peso y estatura
- Membresías: Control de tiempo y renovación
- Nutrición: Agendar citas con nutriólogo
- Subida de contenido: Entrenadores pueden subir videos y PDFs
- Solicitudes: Usuarios premium pueden solicitar rutinas personalizadas

### 🎓 Contexto académico

- **Materia**: Ingeniería de software
- **Temas aplicados**:
  - Desarrollo web con PHP y MySQL
  - Arquitectura cliente-servidor
  - Roles y permisos de usuarios
  - Gestión de contenido multimedia
  - Diseño de interfaces de usuario
  - Base de datos relacional
